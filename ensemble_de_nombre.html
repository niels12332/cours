Ensemble de Nombres

Voici le cours sur un ensemble de nombres :

L'ensemble {0;1;2; 3; ...} est appelé ensemble des nombres entiers naturels. Cet ensemble se note N. L'ensemble {...;-3;-2;-1; 0; 1; 1; 2; 3; ...} est appelé ensemble des nombres entiers relatifs. Cet ensemble se note Z.

  • ce qui veut dire que N ces tous les nombres entiers naturels c'est a dire 0;1;2;3;4...
  • ce qui veut dire que Z ces tous les nombres entiers relatifs c'est a dire ...;-3;-2;-1;0;1;2;3;...
